## 📂 Featured Projects

### 🧠 Intelligent Systems & Machine Learning

#### 1. Reliability-Aware Hierarchical Empathy Classification | [Repo Link](https://github.com/Xawad11/Hierarchical-Empathy-Calibration)
* **Description:** An academic-grade NLP research framework built to classify text-based empathy while prioritizing prediction confidence and uncertainty estimation.
* **Tech Stack:** `PyTorch`, `Transformers (DistilBERT)`, `SciPy`, `Hugging Face`, `CUDA`
* **Key Metrics:** Implemented a multi-stage hierarchical architecture that achieved a **0.72 Macro F1-score**. Integrated post-hoc **Temperature Scaling** via SciPy to reduce Expected Calibration Error (ECE) by **58%**, and deployed **Monte Carlo (MC) Dropout** for inference-time uncertainty quantification.

#### 2. Hybrid Content-Based Movie Recommendation Engine | [Repo Link](https://github.com/Xawad11/Movie-recommendation-system)
* **Description:** An end-to-end information retrieval and data mining pipeline exploring vector spaces, distance metrics, and multi-faceted evaluation paradigms.
* **Tech Stack:** `Python`, `TF-IDF`, `Cosine Similarity`, `KNN`, `K-Means Clustering`, `LSA`, `Support Vector Machines (SVM)`
* **Key Metrics:** Built a hybrid ranking algorithm blending sparse textual similarity vectors with continuous numerical user-vote metrics. Validated search results using industry-standard Information Retrieval parameters: **NDCG@10** and **Precision@10**.

#### 3. AI-Powered Slack Member Analysis Agent | [Repo Link](https://github.com/Xawad11/Slack-AI-Bot)
* **Description:** An autonomous, event-driven background microservice that automatically researches, scores, and profiles incoming Slack community workspace members.
* **Tech Stack:** `Node.js`, `Slack Bolt SDK`, `OpenAI API (GPT-4)`, `PostgreSQL`, `Infrastructure-as-Code (render.yaml)`
* **Key Focus:** Listens asynchronously to platform-level webhooks, enriches metadata using LLMs to calculate automated fit scores, logs telemetry histories into a transactional database layer, and dispatches dynamic digest cards to private channels.
